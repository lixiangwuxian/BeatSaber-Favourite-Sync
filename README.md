# BeatSaber-Favourite-Sync
在不同节奏光剑的PlayerData.dat中同步收藏歌曲。同时支持pc和quest(通过adb或mtp)

需要python3环境，无需额外依赖。

由ChatGPT4-o生成，见<https://chatgpt.com/share/889dea07-94cb-4e7a-af09-c7129a2818fc>
